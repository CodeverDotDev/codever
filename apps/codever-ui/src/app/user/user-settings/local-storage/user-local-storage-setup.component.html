We use local storage to cache public things like tags used for bookmarks and snippets. You can also enable storing some aspects of your data (username and email, private tags and bookmark history) in Local Storage, which will be "cached" for 24h, thus you gain some performance improvements.

Local storage

Please DO NOT store any user related data in local storage (DEFAULT) Store personal data (user details, private tags and bookmark history) in local storage

Local Storage is not considered a secure browser storage. We do our best to protect you against XSS attacks, but an intruder might get access to all the data in the local storage through such an attack. Normally you wouldn't store any sensitive information in your bookmarks or code snippets, so the (minimal) performance gains might still be worth it.

!!! Please enable this option only you are in full control of the device you are running the application on and don't forget to log out if you use the platform on someone else's device !!!
