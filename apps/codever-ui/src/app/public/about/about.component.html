Bookmarking for Developers & Co

Codever is an open source bookmarks code snippets and notes manager created and designed primarily for developers and co with markdown support. We are also building the ultimate curated list of dev resources at https://github.com/CodeverDotDev/bookmarks. You can help by making worthy bookmarks public and rating existing ones. Check our HowTo page to get you started.

We also offer easy access to:
  • History - to easy access your last visited bookmarks (available also via the Ctrl + h shortcut)
  • Read Later - list of bookmarks to read later
  • Pinned - pin the bookmarks you intensively use currently (available also via the Ctrl + p shortcut)
  • Save searches to use later for easier pin pointing known bookmarks

In our public repository of bookmarks and snippets, you can browse lots of good and curated resources from free programming books, to tech blogs covering most programming language and dev podcasts.
