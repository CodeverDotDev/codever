Bookmarklet

Bookmarklets are little javascript links that live in the bookmarks toolbar of your browser. Drag one of the bookmarklet links bellow to your browser's bookmarks to easily save links, code snippets or notes when clicked.

  • - when clicked a popup to save the current URL is displayed [recommended for desktops and notebooks with good resolution]
  • - when clicked it redirects you to the New Bookmark form. Once added, you go back to the initial page [recommended for phones or smaller resolution devices]

You can make a text selection on the page, to use it as description when choosing to create a new bookmark or as code when choosing to create a new snippet out of it.

If no text selection, then the new bookmark dialog is presented. The meta:description, if available, will be used for the description field
