Bookmarking for Developers & Co

Codever is a bookmarks and code snippets manager created and designed primarily for developers and people who work in the information industry. We are also building the ultimate curated list of dev resources at https://github.com/codeverland/bookmarks. You can help by making worthy bookmarks public and rating existing ones. Check our HowTo page to get you started.

Bookmarking and retrieval of bookmarks and code snippets is made easy with the help of:

  • Bookmarklets - drag one of the bookmarklet links mentioned to your browser toolbar
  • Use the Save link to Codever chrome extension for Chromium based browsers - same behaviour as the first bookmarklet
  • Read Later - list of bookmarks to read later
  • Pinned - pin the bookmarks you intensively use currently
  • Favorites - add public bookmarks to favorites to find them in your personal searches
  • Save searches to use later for easier pin pointing known bookmarks
  • Code snippets - special kind of bookmarks where you save small pieces of code to recall later

You can also choose only to browse our public repository of bookmarks. There are lots of good and curated resources from free programming books, to tech blogs covering most programming language and dev podcasts
