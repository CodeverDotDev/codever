Code snippets

Code snippets are special types of bookmarks where you save one or more small pieces of code in the same record.

Save snippets
You can use the Code snippet option in right side of the menu bar
Bookmark Code snippet

or, for an easier experience use one of the following:

Browser Extension

You can use the same browser extension as for saving bookmarks, but you must select the text on web page before triggering the extension :
Chrome Firefox
Bookmarklet
  • - when clicked a popup to save the current URL is displayed [recommended for desktops and notebooks with good resolution]
  • - when clicked it redirects you to the New Bookmark form. Once added, you go back to the initial page [recommended for phones or smaller resolution devices]

You must make a text selection on the page, to use it as as code and choose to create a new snippet when triggering the bookmarklet link.
